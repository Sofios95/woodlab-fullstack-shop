# Woodlab E-shop 🪵

Ένα πλήρες, responsive ηλεκτρονικό κατάστημα (e-shop) για χειροποίητες ξύλινες κατασκευές. Το project υλοποιήθηκε ως μέρος της πρακτικής μου ενασχόλησης μετά την ολοκλήρωση του Web Development Bootcamp.

## 🚀 Τεχνολογίες (Tech Stack)

Το project βασίζεται στο **PERN Stack**:
*   **Frontend:** React.js με Tailwind CSS για το styling.
*   **Backend:** Node.js & Express.
*   **Database:** PostgreSQL.
*   **State Management:** React Hooks & Context API.

## ✨ Χαρακτηριστικά (Features)

*   **Product Catalog:** Δυναμική προβολή προϊόντων από τη βάση δεδομένων.
*   **Shopping Cart:** Πλήρως λειτουργικό καλάθι αγορών (προσθήκη, αφαίρεση, υπολογισμός συνόλου).
*   **Responsive Design:** Βελτιστοποιημένο για χρήση σε όλες τις συσκευές (Mobile, Tablet, Desktop).
*   **Admin Functionality:** Δυνατότητα διαχείρισης αποθέματος και προϊόντων.
